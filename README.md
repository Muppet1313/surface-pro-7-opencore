# Surface Pro 7 - OPENCORE
---
OpenCore configuration for support macOS on Surface Pro 7

### Surface Specs
- Intel® Core™ i5-1035G4 Quad-Core
- 8 GB LPDDR4x RAM
- Intel® Iris™ Plus
- SSD 256 GB
